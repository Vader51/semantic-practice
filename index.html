Today's In-Class Activity

Today we'll practice more with semantic HTML for marking up text, as well as grouping content by topic and using headings to help the user know what the topics on your page are.

Follow the steps below to complete this activity:

  1. Add your name to the meta author tag above.
  2. Open this file in your browser so you can view and test it while we code.
  3. Review this file and its content to get a better idea of what will be on the final page.
  4. Save your file and refresh it in the browser often.

Basic Text Tags

  • p - for paragraph text, usually multiple lines
  • em - for emphasized text
  • strong - for important text
  • mark - marked or highlighted text
  • i - alternative voice text

Special Characters

Examples include:

  • © - copyright
  • ™ - trademark
  •   - non-breaking space
